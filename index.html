1. Why is JavaScript important in web development?

JavaScript is important in web development because it enables dynamic, interactive, and responsive features such as DOM manipulation, event handling, and asynchronous operations.

2. What is the difference between var, let, and const?

The difference between var, let, and const is that var is function-scoped and hoisted with undefined, while let and const are block-scoped and hoisted without initialization, with const also preventing reassignment.

3. What happens if you try to use a variable before declaring it?

Using a variable before declaring it results in undefined for var but throws a ReferenceError for let and const.
